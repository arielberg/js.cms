# CMS-Core Tests

This directory contains tests for the CMS-Core framework.

## Test Structure

- `unit/` - Unit tests for individual modules
- `integration/` - Integration tests for component interactions
- `e2e/` - End-to-end tests for full workflows

## Running Tests

Tests will be implemented using a JavaScript testing framework (to be determined).

## Test Coverage Goals

- Core functionality (moduleLoader, hooks, fixedContentTypes)
- API integrations (GitHubAPI, localNodeJsAPI)
- Content management (contentItem, contentTypeManager)
- Frontend routing (router.mjs)
- Admin panel functionality
